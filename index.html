¿Qué es HTML? es el lenguaje que se utiliza para crear las páginas web a las que se accede mediante internet
Más concretamente, HTML es el lenguaje con el que se "escriben" la mayoría de páginas web.

Las páginas HTML solo uncluían información sobre sus contenidos de texto e imágenes,
tras el paso del tiempo, empezaron a incluir también información sobre el aspecto de sus contenidos,
tipos de letra, colores y márgenes.

¿Qué es CSS?
CSS es un lenguaje de estilos creado para controlar el aspecto o presentación de los documentos electrónicos
definidos con HTML, CSS es la mejor forma de separar los contenidos y su presentación y es indispensable para crear páginas web complejas.

Los Id se utilizan para darle un cambio específico a algún tema, es diferente a una clase ya que la clase,
se le aplican diferentes estilos que comparten características similares, mientras que los IDs se utilizan para identificar y aplicar estilos o manipular algo especófico y único en la página.
